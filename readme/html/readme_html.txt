
----------------Структура HTML документа

<!DOCTYPE html> - первый тег стандарта HTML5
<html> </html>	- контейнер который содержит голову <head> и тело  <body>
<head> </head>	- тег хранения информации о странице
<body> </body>	- тег контета страницы

----------------Теги

<h1>	</h1> -		Заголовок первого уровня (header1-6)
<p>	</p> -		Абзац (paragraph)
<br> -			Тег переноса строки
<hr> -			Тег горизонтальной линии
<em>	</em>		Курсив внути абзаца

----------------Атрибуты

<a>	</a>		Ссылка
href="ссылка"		Ссылка http внутри тега <a>
target="_blank"
disabled		Отключено нажатие
enable			Можно нажать
<buttom>  </buttom>	Кнопка
	Пример
<a href="https://ya.ru" target="_blank">я.ru</a>
<buttom disabled>Нельзя нажимать</buttom>

----------------Списки 

Маркированный-----------------
<ul> </ul> - Тег начала и конца списка (unordered list)
<li> </li> - Пункт списка (list item)

Нумерованный-------------------
<ol> </ol> - Тег начала и конца списка (ordered list)
<li> </li> - Пункт списка (list item)

Список определений--------------
<dl> </dl> - Тег начала и конца списка (discription list)
<dt> </dt> - Термин (term)
<dd> </dd> - Определение (description)

---------------Изображения

<img> - тег, далее вставка атрибутов внутри тега.
src="" - ссылка на картинку
alt="" - текст, который отображается вместо картинки, если она не загрузилась
title="" - текст, который отображается при наведении мыши на картинку
width="" - ширина картинки в пикселях
height="" - высота картинки в пикселях

--------------Сематичные изображения с подписью HTML 5

<figure> </figure> - далее идет вставка тега <img> фигуры
<figcaption> </figcaption> - подпись под лого после <img>

--------------Якоря

<h1 id="anchor">Якорь</h1>
<a href=:#anchor">Переход к якорю</a>

--------------Таблицы

<table> </table> - таблица
<tr> </tr> - строки таблицы (размещение внутри тега)(table row)
<td> </td> - ячейки таблицы (размещение внутри тега)(table data)
<th> </th> - заголовочные ячейки таблицы (table header)
(устарел в html5) <table border="1"> </table> - атрибут границы таблиц

Группировка строк таблиц--------------

<caption> - название таблицы (внутри тега table в самом начале,перед другими тегами)
<thead> - для заголовка
<tfoot> - нижний колонтикул таблицы (использование 1 раз в таблице)
<tbody> - основной контент таблицы (использование сколько угодно раз)
colspan=number - занять количество стобцов в ячейке
rowspan - обьединение строк растягивания, в низ ячейки
