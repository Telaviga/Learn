gcc file.c - где файл имя файла для компиляции
gcc file.c -o file.exe - компиляции и создания файла с расширением .exe
	так же есть доп опции:
	-Wall - выводить базовые предупреждения 
	-Wextra - выводить дополнительные предупреждения
	-Werror - делает все предупреждения ошибками
./file.c - для запуска скомпилируемого файла
